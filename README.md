_StockReader application. Web Service based Stock quote export._

The current version is 0.0.3b. For issues use the [GitHub Issue system](https://github.com/moritzrupp/stockreader/issues).

__This application is still under development.__

##Features

* Comandline based
* Export stocks as CSV with the following information based on the [WebServiceX Stock Quote](http://www.webservicex.net/stockquote.asmx) web service
	* Symbol
	* Last price
	* Date
	* Time
	* Change
	* Open price
	* Highest price
	* Lowest price
	* Volume
	* Market capacity
	* Previous close price
	* Change in percentage
	* Annual range
	* Earns
	* Price/earnings ratio
	* Full name
* Define custom path to CSV file
* Send CSV file by mail to a defined recipient and from a custom email account
* Print only Symbol, Last price, Date, Time and Full name

##Planned Features

* Commandline output of the stocks
* Other file formats then CSV
* Scheduling
